# 🎓 نظام اختبارات القبول - مدارس الأنجال

نظام متكامل لإدارة اختبارات قبول الطلاب في مدارس الأنجال الأهلية والدولية.

---

## 🚀 التشغيل السريع

```bash
# تثبيت المكتبات
npm install

# ملء قاعدة البيانات
npm run seed:full

# تشغيل التطبيق
npm run dev
```

افتح المتصفح على: **http://localhost:3000**
